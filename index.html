What is NPX?

It is a npm package executor

How to use it

It is included in npm version 5.2.0


			npx create-react-app viade_en1b

What it does

  1. Download
  2. Install
  3. Execute
  4. Delete

Advantages

  • Always up-to-date packages
  • Testing CLI tools
